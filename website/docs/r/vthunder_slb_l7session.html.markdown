# vthunder\_slb\_l7session

`vthunder_slb_l7session` Provides details about vthunder SLB l7session
## Example Usage


```hcl
provider "vthunder" {
  address  = "129.213.82.65"
  username = "admin"
  password = "admin"
}

resource "vthunder_slb_l7session" "l7session" {
	sampling_enable {
	    counters1=  "all"
	}
}
```
