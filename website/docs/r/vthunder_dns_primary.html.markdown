# vthunder\_dns\_primary

`vthunder_dns_primary provides` details about Dns Primary
## Example Usage


```hcl
provider "vthunder" {
  address  = "129.213.82.65"
  username = "admin"
  password = "admin"
}

resource "vthunder_dns_primary" "dns_primary" {
		ip_v4_addr = "8.8.8.8"
}
```

## Argument Reference

* `ip_v4_addr` - IP v4 address
